# Mellish-Utility-Scripts
A collection of useful Windows batch files, PowerShell scripts, Python tools, and utility programs created to automate repetitive tasks, organise files, simplify system maintenance, and generally make life easier. Originally built for personal use, but shared for anyone who finds them useful.

Most of these started as solutions to specific problems I encountered and have been cleaned up and shared in case they're useful to others.

## Current Tools

### File Organiser

Automatically sorts files into categories such as:

* Images
* Videos
* Audio
* Documents
* Archives
* Programs
* 3D / Unity / VRChat Assets
* Unknown File Types

Useful for cleaning up messy Downloads folders and project directories.
